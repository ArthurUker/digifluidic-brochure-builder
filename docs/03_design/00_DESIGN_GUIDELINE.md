## 2. 文档目的与适用范围

本文件用于：

1. 定义白皮书/宣传册的整体视觉设计原则；
2. 统一 Web 预览和 PDF 输出的视觉风格；
3. 作为页面布局、打印样式和品牌资产文档的上位依据；
4. 约束组件开发中的样式选择。

本文件不替代：

1. 页面布局规范（`01_PAGE_LAYOUT_SPEC.md`）；
2. 打印样式规范（`02_PRINT_STYLE_SPEC.md`）；
3. 品牌资产使用指南（`03_BRAND_ASSETS_GUIDE.md`）。

## 3. 设计目标

1. **专业可信**：体现生物科技企业的专业性和技术深度；
2. **清晰可读**：确保中英文混排、技术术语和数据的阅读体验；
3. **统一一致**：Web 预览和 PDF 输出视觉风格一致；
4. **打印友好**：充分考虑 A4 打印场景，避免纯屏幕设计；
5. **合规可见**：合规声明在视觉上清晰可辨，不得弱化处理。

## 4. 色彩体系

### 4.1 品牌主色

品牌色以 Digifluidic 品牌识别为基础，具体色值以公司品牌规范为准。建议在 Tailwind 配置中定义以下色阶：

```javascript
// tailwind.config.js 示例
colors: {
  brand: {
    50:  '#e6f7f5',   // 最浅
    100: '#b3e6e0',
    200: '#80d5cc',
    300: '#4dc4b8',
    400: '#26b3a4',
    500: '#00a290',   // 主色（示例色值，以实际品牌色为准）
    600: '#008a7a',
    700: '#007264',
    800: '#005a4e',
    900: '#004238',   // 最深
  },
}
```

### 4.2 功能色

| 用途 | 建议色值 | 说明 |
|---|---|---|
| 正文文字 | `#1a1a1a` 或 `gray-900` | 主文本色 |
| 次要文字 | `#666666` 或 `gray-600` | 说明文字、脚注 |
| 强调色 | 品牌主色 | 链接、重点标注 |
| 成功/正向 | `#16a34a` 或 `green-600` | 可选，用于正向标识 |
| 警告 | `#d97706` 或 `amber-600` | 可选，用于注意事项 |
| 背景色 | `#ffffff` | 白皮书主体背景 |
| 浅灰背景 | `#f5f5f5` 或 `gray-100` | 表格交替行、信息框背景 |
| 分割线 | `#e5e5e5` 或 `gray-200` | 内容分隔 |

### 4.3 使用原则

1. 品牌主色用于封面、章节标题、关键数据标注和装饰元素；
2. 正文使用深灰色而非纯黑色，减少打印对比度过高；
3. 彩色仅用于强调，正文大面积保持黑白灰；
4. 打印时背景色保持可见（`print-color-adjust: exact`）。

## 5. 字体策略

### 5.1 字体选择

| 用途 | 推荐字体 | 备选 |
|---|---|---|
| 中文标题 | 思源黑体 / Noto Sans SC | 系统黑体 |
| 中文正文 | 思源宋体 / Noto Serif SC | 系统宋体 |
| 英文/数字 | Inter 或系统 sans-serif | Arial |
| 代码/数据 | JetBrains Mono 或 Consolas | monospace |

### 5.2 字号层级

| 层级 | Web 字号 | 打印字号 | 用途 |
|---|---|---|---|
| H1（封面标题） | `text-4xl` (~36px) | 28–32pt | 封面主标题 |
| H2（章节标题） | `text-2xl` (~24px) | 18–22pt | 章节起始页标题 |
| H3（小节标题） | `text-xl` (~20px) | 14–16pt | 内容小节标题 |
| H4（子标题） | `text-lg` (~18px) | 12–14pt | 子标题 |
| 正文 | `text-base` (~16px) | 10–11pt | 正文段落 |
| 小字/脚注 | `text-sm` (~14px) | 8–9pt | 脚注、声明、来源 |
| 表格内容 | `text-sm` (~14px) | 8–10pt | 表格单元格 |

### 5.3 行距

- 正文行距：1.6–1.8（`leading-relaxed`）；
- 标题行距：1.2–1.4（`leading-snug`）；
- 列表项间距：适当增加段前段后间距。

### 5.4 字体加载

- 优先使用系统字体，减少外部字体加载；
- 如需使用思源字体，通过 CSS `@font-face` 引入或使用 Google Fonts CDN；
- 打印时确保字体已嵌入 PDF 或使用系统通用字体。

## 6. 图表规范

### 6.1 图片

- 格式：PNG（截图/图表）或 SVG（图标/示意图）；
- 分辨率：至少 150 DPI（打印场景）；
- 颜色模式：RGB（Web 预览）和 CMYK（如打印需要）；
- 图片应有 `alt` 文字和可选的 `caption` 说明。

### 6.2 表格

- 表头使用品牌主色或深灰背景；
- 交替行使用浅灰背景提高可读性；
- 表格宽度适配 A4 页面（约 170mm 可用宽度）；
- 单元格内边距适当（`px-3 py-2`）；
- 表格说明文字置于表格下方。

### 6.3 图表（未来如引入）

- 使用 SVG 或 Canvas 实现，确保打印清晰；
- 配色使用品牌色阶；
- 图例清晰，字号不小于 8pt。

## 7. 间距系统

### 7.1 页面级间距

| 区域 | Web 值 | 打印值 | 说明 |
|---|---|---|---|
| 页面内边距 | `p-8` 或 `p-12` | 15–20mm | A4 页面四周边距 |
| 章节间距 | `my-12` 或 `my-16` | 20–30mm | 章节之间的垂直间距 |
| 段落间距 | `mb-4` 或 `mb-6` | 6–8mm | 段落之间 |
| 标题与正文间距 | `mb-4` | 4–6mm | 标题下方到正文 |

### 7.2 组件级间距

- 使用 Tailwind 默认间距系统（4px 基础单位）；
- 内容区块之间使用 `my-8` 或 `my-12`；
- 列表项之间使用 `mb-2` 或 `mb-3`。

## 8. 封面页设计原则

1. 品牌主色大面积使用或作为装饰元素；
2. 公司/品牌 Logo 置于顶部或中央；
3. 白皮书标题突出显示；
4. 副标题和版本号置于标题下方；
5. 封面底部可放置公司全称和日期；
6. 封面独占一页，触发打印分页。

## 9. 章节页设计原则

1. 章节编号和标题突出；
2. 可使用品牌色装饰条或色块；
3. 章节副标题可选；
4. 章节页独占一页，触发打印分页。

## 10. 合规声明设计原则

1. 合规声明使用独立章节或信息框展示；
2. 字号不得小于正文字号，禁止使用微小字号隐藏声明；
3. 颜色对比度满足可读性要求；
4. 不得使用与背景色相近的颜色弱化声明；
5. 在 PDF 中必须完整保留。

## 11. Web 与 Print 样式隔离

使用 Tailwind CSS 的 `print:` 变体和独立 `print.css` 文件实现样式隔离：

- 屏幕专用样式使用 Tailwind 默认类；
- 打印专用样式使用 `print:` 前缀，如 `print:text-sm`、`print:p-0`；
- 复杂打印样式在 `print.css` 中使用 `@media print` 编写。

## 12. 当前阶段不纳入

1. 响应式移动端设计（白皮书主要为桌面和打印场景）；
2. 暗色模式；
3. 动画和过渡效果（打印场景无意义）；
4. 交互式图表；
5. 自定义品牌字体文件嵌入（可后续补充）；
6. 多语言字体方案。

## 13. 风险与控制措施

| 风险 | 控制措施 |
|---|---|
| 打印效果与屏幕效果差异大 | 开发过程中持续用打印预览检查 |
| 中文字体在 PDF 中缺失 | 使用系统通用字体或确保字体嵌入 |
| 品牌色在打印中失真 | 使用 `print-color-adjust: exact` |
| 合规声明设计弱化 | 在 QA 检查清单中专门检查合规声明样式 |
