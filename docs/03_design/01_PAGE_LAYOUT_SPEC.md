## 2. 文档目的与适用范围

本文件用于：

1. 定义白皮书每页的精确尺寸和布局参数；
2. 统一 Web 预览和 PDF 输出的页面尺寸；
3. 定义页眉、页脚、分页和内容区的布局规则；
4. 指导组件开发中的布局实现。

本文件不替代：

1. 设计规范总则（`00_DESIGN_GUIDELINE.md`）；
2. 打印样式规范（`02_PRINT_STYLE_SPEC.md`）；
3. 品牌资产指南（`03_BRAND_ASSETS_GUIDE.md`）。

## 3. 页面尺寸

### 3.1 基础尺寸

| 参数 | 值 | 说明 |
|---|---|---|
| 页面标准 | A4 | 国际标准 |
| 页面宽度 | 210mm | 固定 |
| 页面高度 | 297mm | 固定 |
| Web 预览容器宽度 | 210mm（约 794px @96DPI） | 居中显示，模拟 A4 |
| Web 预览容器最小高度 | 297mm（约 1123px @96DPI） | 单页最小高度 |

### 3.2 Web 预览实现

```css
/* 页面容器 */
.brochure-page {
  width: 210mm;
  min-height: 297mm;
  margin: 0 auto;
  background: #ffffff;
  box-shadow: 0 0 10px rgba(0, 0, 0, 0.1); /* 仅 Web 预览 */
}
```

### 3.3 打印尺寸

```css
@page {
  size: A4;
  margin: 0; /* 边距由页面内 padding 控制 */
}
```

## 4. 页面边距

### 4.1 标准内容页边距

| 方向 | 值 | 说明 |
|---|---|---|
| 上边距 | 20mm | 页面上方留白 |
| 下边距 | 20mm | 页面下方留白（含页脚区域） |
| 左边距 | 20mm | 页面左侧留白 |
| 右边距 | 20mm | 页面右侧留白 |

### 4.2 封面页边距

| 方向 | 值 | 说明 |
|---|---|---|
| 上边距 | 25mm | 封面留白更大 |
| 下边距 | 25mm | 封面留白更大 |
| 左边距 | 20mm | 与内容页一致 |
| 右边距 | 20mm | 与内容页一致 |

### 4.3 页面容器实现

推荐使用语义化 CSS 类控制 A4 页面尺寸和边距，A4 尺寸、min-height 和页面 padding 主要由 `src/styles/print.css` 或全局 CSS 中的 `.brochure-page` 控制：

```css
.brochure-page {
  width: 210mm;
  min-height: 297mm;
  margin: 0 auto;
  padding: 20mm;
  background: #ffffff;
}
```

```html
<div class="brochure-page">
  <!-- 内容 -->
</div>
```

如需使用 Tailwind 自定义尺寸，应在 `tailwind.config.js` 中定义语义化尺寸（如 `theme.extend.width` 中注册 `'a4': '210mm'`），不建议在组件中大量使用任意值类名（如 `w-[210mm]`）。

## 5. 页眉区域

### 5.1 位置与尺寸

| 参数 | 值 |
|---|---|
| 页眉区域高度 | 10–15mm |
| 页眉距页面上边缘 | 与上边距共用区域 |
| 页眉与正文间距 | 5–8mm |

### 5.2 内容

- 左侧：品牌名或白皮书简称（可选）；
- 右侧：章节标题（可选）；
- 中间：可选装饰线。

### 5.3 打印控制

- 页眉在打印时通过浏览器设置关闭（不在 CSS 中生成）；
- 如需在 PDF 中显示页眉，通过 Playwright 的 `headerTemplate` 参数控制。

## 6. 页脚区域

### 6.1 位置与尺寸

| 参数 | 值 |
|---|---|
| 页脚区域高度 | 10–15mm |
| 页脚距页面下边缘 | 与下边距共用区域 |
| 页脚与正文间距 | 5–8mm |

### 6.2 内容

- 左侧：公司名或版权信息；
- 右侧：页码（如"第 X 页"）；
- 中间：可选版本号。

### 6.3 打印控制

- 页脚在打印时通过浏览器设置关闭（不在 CSS 中生成）；
- 如需在 PDF 中显示页脚，通过 Playwright 的 `footerTemplate` 参数控制；
- Web 预览中页脚作为组件渲染在每页底部。

## 7. 内容区布局

### 7.1 可用内容宽度

| 参数 | 值 |
|---|---|
| 页面总宽度 | 210mm |
| 左边距 | 20mm |
| 右边距 | 20mm |
| 可用内容宽度 | 170mm |

### 7.2 内容区垂直布局

内容从上到下依次排列：

1. 页眉区域（可选）；
2. 章节标题；
3. 章节副标题（可选）；
4. 内容块（段落、列表、图片、表格等）；
5. 行内声明（如有产品参数）；
6. 页脚区域（可选）。

### 7.3 多栏布局

- 当前阶段以单栏为主；
- 如未来需要双栏（如中英文对照），左右栏各约 80mm，中间间距 10mm；
- 表格始终为单栏全宽。

## 8. 分页规则

### 8.1 强制分页

以下位置触发强制分页（`page-break-before: always`）：

| 位置 | 说明 |
|---|---|
| 封面页之后 | 封面独占一页 |
| 目录页之后 | 目录与正文分页 |
| 每个章节起始页 | 新章节从新页开始 |
| 应用矩阵之前 | 大表格从新页开始 |
| 论文证据链之前 | 长列表从新页开始 |
| 合规声明之前 | 合规声明从新页开始 |

### 8.2 禁止分页

以下位置禁止内部分页（`page-break-inside: avoid`）：

| 位置 | 说明 |
|---|---|
| 封面页内部 | 封面不可跨页 |
| 章节起始页内部 | 章节标题和简介不可跨页 |
| 表格行内部 | 单行表格不可跨页（但表格整体可跨页） |
| 图片和其说明 | 图片与 caption 不分离 |
| 合规声明块 | 声明内容尽量在同一页 |

### 8.3 Tailwind 实现

```html
<!-- 强制分页 -->
<div class="page-break-before-always print:page-break-before-always">
  <!-- 新页内容 -->
</div>

<!-- 禁止内部分页 -->
<div class="break-inside-avoid print:break-inside-avoid">
  <!-- 不可拆分的内容 -->
</div>
```

## 11. 特殊页面布局

### 11.1 应用矩阵页

- 表格全宽（170mm）；
- 使用横向页面（可选，但当前阶段保持纵向 A4）；
- 如表宽超出，可适当缩小字号或使用 `landscape` 打印。

### 11.2 论文证据页

- 每篇论文作为一个条目；
- 条目内不可分页；
- 分组标题醒目。

### 11.3 合规声明页

- 独立章节，独占页面或从新页开始；
- 声明文字清晰可读；
- 底部可添加公司盖章区（预留空白）。

## 12. 当前阶段不纳入

1. 横向 A4 页面布局；
2. 多栏文字布局；
3. 响应式布局（当前仅适配桌面/打印）；
4. 交互式页面导航（如侧边栏目录）。

## 13. 风险与控制措施

| 风险 | 控制措施 |
|---|---|
| Web 预览与打印尺寸不一致 | 使用绝对单位 mm，避免相对单位 |
| 分页位置不合理 | 使用 `break-inside: avoid` 保护关键内容，QA 时逐页检查 |
| 长表格跨页截断 | 表头使用 `thead` + CSS `display: table-header-group` 在每页重复 |
| 图片超出页面宽度 | 设置 `max-width: 100%`，图片宽度不超过 170mm |
