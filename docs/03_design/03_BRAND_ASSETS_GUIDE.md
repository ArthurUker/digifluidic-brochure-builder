## 2. 文档目的与适用范围

本文件用于：

1. 统一白皮书/宣传册中品牌元素的使用方式；
2. 约束 Logo 的放置位置、尺寸和周围留白；
3. 确保产品名称和品牌表述的一致性；
4. 防止品牌资产的不当使用。

本文件不替代：

1. 公司正式品牌识别手册（Brand Guidelines）；
2. 设计规范总则（`00_DESIGN_GUIDELINE.md`）；
3. 页面布局规范（`01_PAGE_LAYOUT_SPEC.md`）。

## 3. 品牌名称使用

### 3.1 品牌名

| 项目 | 正确写法 | 说明 |
|---|---|---|
| 品牌名 | **Digifluidic** | 首字母大写，其余小写 |
| 中文公司名 | **珠海市迪奇孚瑞生物科技有限公司** | 完整法定名称 |
| 英文公司名 | **Zhuhai Digifluidic Bio-tech Co., Ltd.** | 完整英文法定名称 |
| 品牌简称 | **Digifluidic** | 不使用"DF"等缩写 |

### 3.2 禁止写法

- ~~DigiFluidic~~（F 大写）
- ~~digifluidic~~（全小写）
- ~~DIGIFLUIDIC~~（全大写，除非 Logo 本身如此）
- ~~迪奇孚瑞~~（对外正式材料中不建议单独使用"迪奇孚瑞"，应使用公司全称"珠海市迪奇孚瑞生物科技有限公司"或品牌名 Digifluidic。内部讨论或内部文档中可使用简称，但对外发布材料必须使用全称或品牌名。）

## 4. 产品名称使用

### 4.1 核心产品

| 产品 | 正确名称 |
|---|---|
| 全自动核酸分析仪 | **VHunter Plus** 全自动核酸分析仪 |

首次出现时使用完整名称："VHunter Plus 全自动核酸分析仪"，后续可简称为"VHunter Plus"。

### 4.2 其他产品/材料

| 类别 | 命名规则 |
|---|---|
| 微流控芯片 | "XX 检测芯片"，如"食源性病原体检测芯片" |
| 检测方案 | "XX 检测方案"，如"SARS-CoV-2/流感检测方案" |
| 试剂耗材 | 以公司正式产品名称为准 |

### 4.3 产品名禁止写法

- 不得在产品名中添加未经确认的修饰词（如"革命性""划时代"）；
- 不得在产品名中使用注册标识（®、™）除非已在对应地区完成注册；
- 不得虚构不存在的产品型号或规格。

## 5. Logo 使用规范

### 5.1 放置位置

| 页面 | 位置 | 说明 |
|---|---|---|
| 封面页 | 顶部居中或左上角 | 最醒目的位置 |
| 内容页页眉 | 左侧 | 小尺寸 Logo |
| 合规声明页 | 顶部 | 正式文档标识 |

### 5.2 尺寸建议

| 场景 | 建议尺寸 |
|---|---|
| 封面 Logo | 宽度 60–80mm |
| 页眉 Logo | 宽度 20–30mm |
| 页脚 Logo | 宽度 15–20mm |

### 5.3 周围留白

- Logo 四周至少保留 Logo 高度 50% 的留白空间；
- 不得将文字、图表或其他元素紧贴 Logo。

### 5.4 Logo 文件

- 格式：SVG（优先）或 PNG（透明背景）；
- 存放路径：`public/brand/logo.svg`；
- 颜色：优先使用品牌色版本，如需在深色背景上使用可提供反白版本。

### 5.5 禁止行为

- 不得拉伸、压缩或变形 Logo；
- 不得更改 Logo 颜色（除使用正式的反白版本外）；
- 不得在 Logo 上叠加文字或图形；
- 不得使用低分辨率或模糊的 Logo 图片；
- 不得将 Logo 用作背景水印（除非设计明确需要）。

## 6. 品牌色使用

### 6.1 色值定义

品牌色以公司正式品牌规范为准。在设计文件和代码中使用时，通过 CSS 变量或 Tailwind 配置集中定义，确保全局一致。

### 6.2 使用场景

| 场景 | 品牌色使用方式 |
|---|---|
| 封面背景 | 可大面积使用品牌主色 |
| 章节标题 | 品牌主色或深色变体 |
| 关键数据 | 品牌主色标注 |
| 装饰线/色块 | 品牌主色或浅色变体 |
| 链接 | 品牌主色 |
| 表格表头 | 品牌主色背景 + 白色文字 |

### 6.3 禁止行为

- 不得在正文大面积使用品牌色（影响阅读）；
- 不得使用未经品牌规范确认的颜色替代品牌色；
- 不得在合规声明中使用弱化颜色。

## 7. 品牌字体

品牌字体以公司正式品牌规范为准。在白皮书设计中：

- 中文标题：建议使用思源黑体或无衬线字体；
- 中文正文：建议使用思源宋体或衬线字体；
- 英文/数字：建议使用 Inter 或系统 sans-serif。

如公司有指定品牌字体，以公司规范为准。

## 8. 品牌表述规范

### 8.1 公司简介标准表述

> 珠海市迪奇孚瑞生物科技有限公司（Digifluidic）是一家专注于数字微流控核酸检测技术的高科技企业，致力于为食品安全、临床诊断和公共卫生领域提供自动化、多靶标的核酸检测解决方案。

### 8.2 平台描述标准表述

> Digifluidic 数字微流控核酸检测平台以 VHunter Plus 全自动核酸分析仪为核心硬件，结合微流控芯片和试剂耗材体系，实现从样本到结果的自动化核酸检测流程。

### 8.3 禁止表述

- 不得使用"行业第一""全球领先"等绝对化表述；
- 不得在未获授权的情况下声称与某机构的合作关系；
- 不得使用未经公司确认的愿景或使命表述。

## 9. 品牌资产文件管理

### 9.1 推荐目录结构

```text
public/
└── brand/
    ├── logo.svg              # 品牌 Logo（主版本）
    ├── logo-white.svg        # 反白 Logo（如需要）
    ├── logo-icon.svg         # 图标版 Logo（如需要）
    └── README.md             # 品牌文件说明
```

### 9.2 文件来源

品牌资产文件（Logo、字体等）应由公司设计或市场部门提供正式版本，不得从网页截图或二次加工获取。

### 9.3 版本控制

- 品牌资产文件纳入 Git 版本控制；
- Logo 更新时需同步更新所有引用；
- 旧版本 Logo 文件应替换而非并存。

## 10. 当前阶段不纳入

1. 品牌识别手册全文（仅引用关键规则）；
2. 品牌视频或动画资产；
3. 多语言品牌表述（当前仅中文）；
4. 社交媒体品牌资产（头像、封面等）；
5. 印刷品专用品牌资产（专色等）。

## 11. 风险与控制措施

| 风险 | 控制措施 |
|---|---|
| 品牌表述与公司正式口径不一致 | 所有品牌表述以本文件为准，变更需经公司确认 |
| Logo 文件过时或非正式版本 | 定期与设计/市场部门确认 Logo 文件为最新正式版本 |
| 产品名称拼写错误 | 本文件集中定义所有产品名称，组件中引用而非硬编码 |
| 品牌色色值不一致 | 通过 Tailwind 配置和 CSS 变量集中定义 |

1. 公司品牌规范更新后，第一时间同步更新本文件；
2. Logo 文件变更后，重新生成 PDF 检查所有页面的 Logo 显示；

4. 品牌表述变更需记录变更原因和审批人。
