## 3. 运维范围

### 3.1 本系统运维范围

| 运维对象 | 说明 |
|---|---|
| Nginx 服务 | 确保 Nginx 正常运行，8083 端口可访问；reload 前必须执行 `nginx -t` 测试配置，通过后再执行 `nginx -s reload` |
| 静态文件 | `C:\digifluidic-brochure\dist\` 目录完整性 |
| 页面访问 | 浏览器访问 `http://服务器IP:8083` 正常 |
| PDF 导出 | 定期验证 PDF 导出功能正常 |
| 备份 | 定期检查备份文件完整性 |

### 3.2 不在运维范围

- RDPMS 系统；
- 食品安全检验管理系统；
- 服务器操作系统维护；
- 网络和防火墙（仅确认 8083 端口可达）；
- 数据库；
- PM2 进程。

## 4. 日常检查

### 4.1 检查频率

| 检查项 | 频率 | 执行方式 |
|---|---|---|
| 页面可访问性 | 每日 | 浏览器访问 |
| Nginx 进程状态 | 每周 | 远程桌面检查 |
| 端口监听状态 | 每周 | 命令行检查 |
| 磁盘空间 | 每月 | 远程桌面检查 |
| 备份完整性 | 每月 | 检查备份目录 |

### 4.2 页面可访问性检查

1. 浏览器访问 `http://服务器IP:8083`；
2. 确认页面正常加载；
3. 随机点击或滚动检查几个章节；
4. 确认合规声明可见。

### 4.3 Nginx 状态检查

在服务器上以管理员身份打开 PowerShell：

```powershell
# 检查 Nginx 进程
Get-Process nginx

# 检查 8083 端口监听
netstat -ano | findstr :8083
```

预期输出：
- Nginx 进程存在（通常 1 个 master + 多个 worker）；
- `:8083` 处于 LISTENING 状态。

### 4.4 磁盘空间检查

```powershell
# 检查 C 盘剩余空间
Get-PSDrive C
```

确保剩余空间 > 1GB（静态站点占用很小，但需留有备份空间）。

### 4.5 备份完整性检查

```powershell
# 检查备份目录
Get-ChildItem C:\digifluidic-brochure\backup

# 确认最新备份包含 index.html
Test-Path C:\digifluidic-brochure\backup\<最新备份目录>\index.html
```

## 5. 故障处理流程

### 5.1 故障发现

- 用户反馈页面无法访问；
- 日常检查发现异常；
- 服务器监控告警（如有）。

### 5.2 标准处理流程

```
发现故障
  ↓
1. 确认故障现象
   - 页面无法访问？404？500？
   - 是所有页面还是特定页面？
  ↓
2. 初步排查
   - 检查 Nginx 是否运行
   - 检查 8083 端口是否监听
   - 检查 dist/ 目录是否完整
  ↓
3. 根据故障类型处理
   - 参见 01_COMMON_ISSUES.md
  ↓
4. 如无法快速修复 → 回滚
   - 参见 02_ROLLBACK_GUIDE.md
  ↓
5. 修复后验证
   - 浏览器访问确认
  ↓
6. 记录故障
   - 故障时间、现象、原因、处理方式
```

### 5.3 紧急联系方式

| 角色 | 联系时机 |
|---|---|
| 技术负责人 | 无法自行解决的故障 |
| 服务器管理员 | Nginx 无法启动、端口冲突等系统级问题 |
| 内容负责人 | 内容显示异常 |

## 6. 内容更新运维

### 6.1 文案修正

如果仅修改白皮书中的文字内容（不改代码）：

1. 在本地编辑 `src/data/brochure.ts`；
2. 本地 `npm run dev` 预览确认；
3. `npm run build` 构建；
4. 上传 `dist/` 到服务器替换；
5. 重载 Nginx（如修改了 Nginx 配置）或无需重载（仅替换静态文件）；
6. 浏览器验证。

### 6.2 图片更新

1. 将新图片放入 `public/` 目录；
2. 更新 `src/data/brochure.ts` 中的图片引用；
3. 构建 → 部署 → 验证。

## 7. 监控建议

当前阶段不要求自动化监控，但建议定期执行以下手动检查：

| 检查项 | 方法 |
|---|---|
| 页面可访问 | 浏览器访问 |
| 页面加载速度 | 浏览器 DevTools Network 标签 |
| 死链检查 | 手动点击所有锚点链接 |
| PDF 导出 | 浏览器打印 → 另存为 PDF |
