## 2. 文档目的与适用范围

本文件用于：

1. 提供 Chrome 浏览器手动打印 PDF 的完整操作步骤；
2. 作为 Playwright 自动导出失败时的兜底方案；
3. 指导非技术人员进行手动 PDF 导出。

本文件不替代：

1. PDF 导出概览（`00_PDF_EXPORT_OVERVIEW.md`）；
2. Playwright 导出规范（`01_PLAYWRIGHT_EXPORT_SPEC.md`）。

## 3. 适用场景

- Playwright 自动导出不可用时；
- 需要快速预览打印效果；
- 非技术人员需要导出 PDF；
- 开发过程中调试打印样式。

## 4. 操作步骤

### 步骤 1：启动页面

**方式一：开发服务器**

```bash
npm run dev
```

浏览器访问 `http://localhost:5173`

**方式二：预览服务器（推荐，与部署效果一致）**

```bash
npm run build
npm run preview
```

浏览器访问 `http://localhost:4173`

**方式三：已部署的服务器页面**

浏览器访问 `http://服务器IP:8083`

### 步骤 2：打开打印对话框

- **macOS**：`Cmd + P`
- **Windows/Linux**：`Ctrl + P`

或者：浏览器菜单 → 打印

### 步骤 3：配置打印参数

| 参数 | 设置值 | 说明 |
|---|---|---|
| 目标打印机 | **另存为 PDF** | 不要选择实体打印机 |
| 页面尺寸 | **A4** | 210mm × 297mm |
| 边距 | **无** | 页面边距由 CSS 控制 |
| 背景图形 | **开启** ✓ | 确保品牌色和背景图可见 |
| 页眉和页脚 | **关闭** ✗ | 不打印 URL、页码等浏览器默认信息 |
| 缩放 | **默认**（100%） | 不缩放 |

### 步骤 4：预览检查

在打印对话框中逐页预览：

1. 封面页是否完整；
2. 分页位置是否合理；
3. 表格是否被截断；
4. 图片是否正常显示；
5. 合规声明是否完整。

### 步骤 5：保存 PDF

1. 点击"保存"按钮；
2. 选择保存位置；
3. 文件名按规范命名：
   ```
   Digifluidic_Brochure_customer_zh-CN_v2026.06.pdf
   ```
4. 点击"保存"。

### 步骤 6：验证 PDF

打开保存的 PDF 文件，按 `03_PDF_QA_CHECKLIST.md` 逐项检查。

## 6. 常见问题

### 6.1 打印预览显示空白

- 确认页面在浏览器中正常显示（非打印模式）；
- 检查 `@media print` 是否误隐藏了内容；
- 刷新页面后重试。

### 6.2 品牌色不显示

- 确认"背景图形"选项已勾选；
- 检查 Chrome 版本（较旧版本可能不支持）；
- 使用 Chrome 最新版本。

### 6.3 页眉页脚仍然显示

- 确认"页眉和页脚"选项已取消勾选；
- 该选项在"更多设置"中。

### 6.4 字体不正确

- 确认系统已安装所需字体；
- 检查页面 CSS 的 `font-family` 是否正确；
- Chrome 打印使用系统字体，确保字体已安装到操作系统。

### 6.5 分页位置与预期不符

- 检查 `page-break-before` 和 `page-break-after` CSS 规则；
- 某些元素可能因为 `break-inside: avoid` 导致意外分页；
- 调整元素高度或分页策略。

## 7. 不同浏览器差异

| 浏览器 | 打印兼容性 | 备注 |
|---|---|---|
| Chrome | ★★★★★ | 最佳兼容性，推荐使用 |
| Edge | ★★★★★ | 与 Chrome 内核相同 |
| Firefox | ★★★★ | 部分打印样式可能有差异 |
| Safari | ★★★ | 打印样式支持有限 |

**推荐**：始终使用 Chrome 进行打印导出和验证。
