## 2. 文档目的与适用范围

本文件用于：

1. 提供 PDF 导出后的标准化检查流程；
2. 确保每次导出的 PDF 质量一致；
3. 作为发布前必须通过的检查清单。

本文件不替代：

1. PDF 导出概览（`00_PDF_EXPORT_OVERVIEW.md`）；
2. Playwright 导出规范（`01_PLAYWRIGHT_EXPORT_SPEC.md`）；
3. 内容评审文档（`docs/07_review_release/00_INTERNAL_REVIEW_GUIDE.md`）。

## 4. QA 检查流程

### 4.1 标准流程

```
1. 导出 PDF
2. 打开 PDF 文件
3. 按本清单逐项检查
4. 记录不通过项
5. 修复问题
6. 重新导出 PDF
7. 重新按清单检查
8. 全部通过 → QA 完成
```

### 4.2 快速检查（小改动后）

如果仅修改了内容数据（如文案修正），可执行简化检查：

- 文件级检查（3.1）
- 内容完整性检查（3.9）
- 合规声明检查（3.8）
- 抽查 3–5 个关键页面

### 4.3 完整检查（大改动后）

如果修改了样式、布局、分页或组件，必须执行完整检查。

## 6. 严重程度定义

| 级别 | 定义 | 处理方式 |
|---|---|---|
| 高 | 影响合规、品牌形象或内容完整性 | 必须修复，阻塞发布 |
| 中 | 影响阅读体验或美观度 | 建议修复，不阻塞发布 |
| 低 | 微小瑕疵，不影响使用 | 可记录后择期修复 |
